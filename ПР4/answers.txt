1. Flexbox меняет способ вычисления размеров и расположения элементов. Дочерние элементы становятся flex-элементами и выстраиваются вдоль основной оси.

2.

justify-content выравнивает элементы по основной оси (например, space-between).

align-items выравнивает элементы по поперечной оси (например, center).

3. flex-direction задает направление основной оси. Значения: row (по умолчанию), row-reverse, column, column-reverse. Меняет направление основной и поперечной осей.

4. flex: 1 = flex-grow: 1 (элемент растет), flex-shrink: 1 (сжимается), flex-basis: 0% (начальный размер 0).

5. flex-wrap: wrap разрешает перенос элементов на новую строку. Используется когда элементы не помещаются в контейнере.

6.

align-items - для одной строки, выравнивает элементы по поперечной оси

align-content - для многострочного контейнера, распределяет строки

7. order меняет порядок отображения элементов. Полезно для адаптивных макетов и изменения порядка без перестановки в HTML.

8. Основная ось - главное направление (задается flex-direction). Поперечная ось - перпендикулярна основной.

9. Использовать flex-wrap: wrap с flex-basis. Пример:

css
.container { display: flex; flex-wrap: wrap; }
.item { flex: 1 1 300px; }

10. gap создает равные промежутки только между элементами (не по краям), упрощает код и избегает лишних отступов.
